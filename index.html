// 02. Projects

Things I've Built

HPC filetransfer2026

An HPC file-transfer benchmarking suite that times, monitors CPU/RAM/network, and checksum-verifies transfers between cluster nodes across both many-small-file and single-huge-file workloads. Includes MSC and MCD — two multithreaded, multi-socket copy tools written in C — benchmarked against scp and dd through a SLURM-driven harness.

C Python HPC / SLURM Sockets

Android Static Analysis

Androguard-based static analysis of 199 real-world Android APKs for TLS certificate-validation misuse, over-privileged and unused permissions, and unguarded exported components (CWE-295, -250, -926; OWASP MASVS). Every automated finding is run through LLM triage and a manual sample, then reported with a per-check precision rate — the study's real result is where scanners are trustworthy versus where they just generate alert fatigue. Co-authored research paper included.

Python Androguard Android AppSec Static Analysis CWE / OWASP

Cyber Bridge VR

A low-fidelity product concept for an immersive VR cybersecurity-awareness platform that drops users inside realistic phishing and social-engineering scenarios to build threat-recognition instincts in real time — targeting the ~83% of breaches that trace back to human error, where slide-and-quiz training falls short. Built with a William & Mary team for the Commonwealth Cyber Initiative's Innovate Cyber Challenge, where it won $500 (Spring 2026).

Security Awareness Lo-fi Prototype Product Design
